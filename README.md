# 小兔鲜

## 技术栈

```
Vue3 ts vive vue-router pinia(大菠萝) axios element-plus vue-lazyload
```

**搭建工具**：Vite

## 启动项目

```
cd vite-project

npm install

npm run dev
```

## 项目介绍

- 功能：登录 购物车 会员中心 我的订单 商品详情页 

- 涉及：本地购物车，用户购物车 

- 性能优化：图片懒加载 路由懒加载 pinia优化二次请求
